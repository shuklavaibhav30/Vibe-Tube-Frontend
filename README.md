## Overview

VibeTube is a YouTube-inspired full-stack video sharing platform built using React, Node.js, Express.js, MongoDB, JWT Authentication, and Cloudinary.

This repository contains the frontend application where users can browse videos, upload content, manage playlists, subscribe to channels, interact through comments and tweets, and manage their profiles.

## Installation

Clone the repository:

```bash
git clone https://github.com/shuklavaibhav30/Vibe-Tube-Frontend.git
```

Move into project:

```bash
cd FRONTEND
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
